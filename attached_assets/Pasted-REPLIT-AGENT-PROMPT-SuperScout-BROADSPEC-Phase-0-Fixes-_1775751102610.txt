FIX 2 — Restrict CORS on the API server
In artifacts/api-server/src/app.ts, the line app.use(cors()) allows requests from any website. Replace it with a restricted version:
app.use(cors({
  origin: [
    'https://superscout.pro',
    process.env.CORS_ALLOWED_ORIGIN || 'http://localhost:3000'
  ]
}));
This limits API access to the SuperScout domain and the local development server. The CORS_ALLOWED_ORIGIN environment variable lets Replit's preview URL be added without hardcoding it.
